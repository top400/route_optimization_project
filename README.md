# Smart Route Optimization in a City Using Multi-Weighted Graphs

## Project Overview
This project studies route optimization in urban road networks using graph algorithms.
Instead of using only distance, each road segment includes multiple attributes such as traffic, safety, and road quality.

## Mathematical Topics
- weighted graphs
- shortest path algorithms
- multi-criteria optimization
- linear cost functions

## Implementation
The project uses Python to:
- model an urban graph
- compute route costs
- run Dijkstra's algorithm
- compare different routing scenarios

## Experiments
- shortest route by distance
- traffic-aware route
- safety-aware route
- balanced route

## Conclusion
The project shows that different routing priorities lead to different optimal paths, making the model more realistic than standard shortest path examples.
